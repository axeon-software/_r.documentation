# Multi Material

[doc.bjs.com - How To use Multi Materials](http://doc.babylonjs.com/how_to/multi_materials)

```json
[
    {
    "mesh3" : {
        "material" : {
          "subMaterials" : {
            "0" : {
              "diffuseColor" : "yellow"
            },
            "1" : {
              "diffuseColor" : "white"
            }
          }
        }
      }   
    }   
]
```
